# Lumina - Student Study Group Platform

Lumina is a web-based platform designed to help students connect and collaborate by forming or joining study groups. This project was developed as part of a Web Development course to demonstrate full-stack integration and UI/UX design.

## 🚀 Live Demo
View the live project here: [https://lumina-uw.web.app/](https://lumina-uw.web.app/)

## ✨ Features
- **User Authentication:** Secure login and signup powered by Firebase Auth.
- **Dynamic UI:** Role-based interface changes using JavaScript and the Bulma CSS framework.
- **Group Management:** Real-time ability to create, search for, and join study groups.
- **Database Integration:** Persistent storage for user profiles and group data using Firebase Firestore.
- **Automated Deployment:** Integrated CI/CD pipeline via GitHub Actions for seamless updates to Firebase Hosting.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, [Bulma](https://bulma.io/), JavaScript (ES6+)
- **Backend/Hosting:** [Firebase](https://firebase.google.com/) (Hosting, Auth, Firestore)
- **CI/CD:** GitHub Actions
